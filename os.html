Ubuntu入門

土井　翔平

2018年6月16日

ここでは代表的なOSを比較した上で、Linuxの一種のUbuntuを研究目的に利用するための準備を解説します。 なお、環境はASUS Zenbook 3 (UX390UA) でUbuntu 18.04 LSTになります。

OSの比較

ここでは代表的なパソコン用OSとしてWindows, Mac, Linuxを比較します。 ただし、僕はMacを使ったのははるか昔であり、Windowsは特に好きではないという偏見が含まれています。

Windows

世間的に見て事実所のデファクトスタンダードなので、大抵のソフトは動きますし、Windows前提に解説されていることが多いです。 Windowsを使っていて不便や不満を感じない人は特に他のOSを使う必要はないかと思います。

潜在的な問題点としては制御が困難なアップデートがある、文字コードがUTF8ではないのでテキスト分析で面倒が生じる（僕がLinuxに乗り換えたきっかけはこれです）、CUIよりもGUIベース、デザインがなんとなくダサい、などがあります。 また、体感ではありますが、Linuxのほうが電力消費が少なく、処理も早いように思います。

Mac

最近、研究界隈でも利用している人が非常に多い印象です。 僕が使っていた頃はWindowsでは動くけどMacでは（正常に）動かないソフトが多かった気がしますが、今ではそんなことはないのではないかと思います。 Windowsに比べて洗練されたデザインのイメージがありますし、iPhoneやiPadを使っている人は連携機能もあるのではないでしょうか。 他にも、いろいろ便利な帰納があると風のうわさでは聞いています。 Linuxも結局はmacと同じUNIXから派生しているので、CUIによる管理も楽ですし、文字コードの問題もありません。 Windowsは嫌いだけどMacが好きな人はMacがいいと思います。

Macのありうる問題としては同性能のWindowsマシン比べて高いというのがありますが、そもそもOSが違いますし、デザインや部品も違うでの一概には言えないような気がします。 むしろ、個人的にはMac製品しか選択肢がない点にやや不満を覚えます。 例えば、1kg以下ではMacBookしか選択肢はなく、それなりの性能を求めると1kgを超えるMacBook Proにせざるを得ません。

Linux

Linuxの魅力は無料である点です。 そのため、様々なOSから選択をすることができ、その上でデザインのカスタマイズ性も高く、好みのPCにすることができます。 また、CUIで管理できるのでいろいろと便利です。 すでに述べたように、Windowsよりは動作が軽く、省エネな気がします。 また、Linuxは基本的にWindowsマシンか自作PCで動かすので、ハードウエアの選択肢も非常に豊富です。

Linuxの最大の問題は情報量の少なさにあります。 有名なOSではコミュニティなどでいろいろな情報が出回っていますが、最終的には自力で（多くの場合は英語の）情報を見つけてくる必要があります。 当然、メーカーによるサポートもないので、OSを入れてもうまく動かないこともあります。 また、OfficeのようにWindowsやMacはサポートしているけどLinuxでは動かないソフトも山ほどあります。

にもかかわらず、Linuxを利用する理由は、WindowsもMacもあまり好きではないという個人的事情に加えて、自分の好きなマシンに好きなOSを入れて好きなデザインを使えるという自由さになります。

Linuxを使うには

とはいいつつも、現実的にLinuxだけというのは難しいので（OfficeやInternet Explorerと無縁に過ごすのは不可能でしょう）、Windowsと併用することを前提に進めていきます。 大きく分けると2つあり、WindowsとLinuxを同じマシンに別々に共存させるマルチブートと、WIndows上であたかもソフトの1つのようにLinuxを用いる仮想マシンです。
